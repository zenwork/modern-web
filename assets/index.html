This project setup is a good starting point for non-trivial web-based libraries or app projects written with Typescript and Lit-Element. It integrates all the great work coming out of the modern-web, open-wc, and Polymer Project communities. Dev & Test

Combines Typescript, Lit-Element, and Css-in-Css with es-web-dev's es6 module reloading.

Run a single suite of tests in multiple browsers with mocha & playwright.

Typescript maps make in-browser and in-IDE debugging, in test and in production code an easy thing!

 Build

Use Roll-up to bundle everything for production

Uses a set of shared Roll-up plugins across live-coding, testing, and bundling

 Documentation

Use mdjs instead of storybook to demo and document components

Generate web component descriptors

 This project can be adapted for browser libraries or for web-sites/app development.
